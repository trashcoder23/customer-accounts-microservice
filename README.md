
# Customer Accounts Microservice

A RESTful microservice for managing customer accounts with CI/CD pipeline, Docker containerization, and Kubernetes deployment.

## Build Status

[![CI/CD Pipeline](https://github.com/trashcoder23/customer-accounts-microservice/actions/workflows/ci-build.yaml/badge.svg)](https://github.com/trashcoder23/customer-accounts-microservice/actions/workflows/ci-build.yaml)

## Features

- Create, Read, Update, Delete customer accounts
- RESTful API design
- Automated testing with nose
- CI/CD with GitHub Actions
- Docker containerization
- Kubernetes deployment
- Security headers with Talisman
- CORS policies
